Делаем галочки для выбора режима отрисовки:
Метод отрисовки вызывает в классе RenderEngWithTriangFill на 90 строке туда,
в качестве параметра передаётся массивы с координатами, цвет, текстура, список с текстурными вершинами зет буфер и флаг,
который отвечает за отрисовку полигональной сетки, если флаг false, тo сетку мы не рисуем. у нас по умолчанию рисуется  либо сплошной цвет
либо текстура, если флаг true, то будет рисоваться полигональная сетка. Т.е. галочка "Рисовать полигональную сетку"
меняет значение это флага (где и откуда оно буде приходить в мой метод тебе придется подумать самому)

Чтобы сработала галочка наложить текстуру на модель, нам нужно в этом классе RenderEngWithTriangFill (пока 33 строка)
присвоить переменной текстуры значение из соответствующего поля, класса модели и тогда уже передать в мой метод для
растеризации. Есть ли у нас выбрана галочка залить все сплошным цветом (ее нужно ставить по умолчанию), то поле
для текстуры должно оставаться пустым. Для этого случая наверное тебе нужно будет в классе RenderEngWithTriangFill написать
условие, которое будет считывать выбрана ли галочка наложить текстуру и в зависимости от значения уже обращаться к полю
у модели, чтоб получить эту текстуру. В общем здесь с флагами полная свобода действий. Можешь дополнять мои методы
необходимыми параметрами, если это нужно, главное логику не трогать, а то я сам не до конца понимаю как она работает:)
